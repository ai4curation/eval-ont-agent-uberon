# PR Summary: Add Disjoint Axiom Between Multicellular Organism and Organism Substance

## Overview
This PR implements the resolution to issue #2421 by adding a disjoint axiom between UBERON:0000468 (multicellular organism) and UBERON:0000463 (organism substance).

## Rationale
As discussed in the issue, there was no disjunction between these two fundamental concepts in UBERON:
- **UBERON:0000468 (multicellular organism):** "Anatomical structure that is an individual member of a species and consists of more than one cell."
- **UBERON:0000463 (organism substance):** "Material anatomical entity in a gaseous, liquid, semisolid or solid state; produced by anatomical structures..."

Without this disjoint axiom, it was theoretically possible to create an inconsistent definition such as:
- "human milk" ≡ milk AND Homo sapiens

This would incorrectly classify an organism substance as a multicellular organism, creating a logical contradiction.
