# Fix tracheal mucosa logical definition (issue #3617)

## Summary

This PR fixes an erroneous inferred parent-child relationship reported in [issue #3617](https://github.com/obophenotype/uberon/issues/3617): UBERON:0001826 *nasal cavity mucosa* was being inferred as a subclass of UBERON:0000379 *tracheal mucosa*. The original issue was raised from the Human Disease Ontology side.

## Root cause

UBERON:0000379 *tracheal mucosa* was equivalent to:

```
mucosa and part_of some respiratory airway   (UBERON:0001005)
```

UBERON:0001826 *nasal cavity mucosa* is defined as:

```
mucosa and part_of some olfactory apparatus chamber   (UBERON:0015788)
```

Because the olfactory apparatus chamber is (correctly) part of the respiratory tract / airway system, any mucosa part-of the olfactory apparatus chamber satisfies the over-general definition of tracheal mucosa, producing the unwanted subclass inference.

## Fix

As suggested in the issue thread by @dosumis, I changed the `part_of` filler in the equivalence axiom of UBERON:0000379 from *respiratory airway* (UBERON:0001005) to *trachea* (UBERON:0003126). The text definition was updated in parallel so it mirrors the new logical definition.

Diff on `src/ontology/uberon-edit.obo`:

```
-def: "A mucosa that is part of a respiratory airway." [OBOL:automatic]
+def: "A mucosa that is part of the trachea." [OBOL:automatic]
...
-intersection_of: part_of UBERON:0001005 ! respiratory airway
+intersection_of: part_of UBERON:0003126 ! trachea
```
