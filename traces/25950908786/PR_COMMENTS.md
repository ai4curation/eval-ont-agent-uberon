### Rationale

The old definition explicitly excluded any complete cells as parts, which caused problems for many current subclasses—especially nervous-system regions such as neuropils and cell body rinds that routinely contain whole glial cells alongside partial neuronal projections. FlyBase already addressed the same problem in `FBbt:00007060` by broadening its definition; making the parallel change in Uberon keeps the two ontologies aligned and removes an implicit, biologically inaccurate exclusion.

The logical axiom `composed_primarily_of GO:0005575 ! cellular_component` already captures the intended meaning ("mainly cell components"), so no logical changes were required—only the textual definition needed updating.
