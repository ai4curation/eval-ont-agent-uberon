This PR modifies the logical definition of **squamous epithelium** (UBERON:0006914) to address issue #3473.

### Problem

The existing logical definition used `has_part some squamous epithelial cell`. This was overly broad: any epithelium containing even a single squamous epithelial cell (or a minority population) would be inferred as a subclass of squamous epithelium, which contradicts standard anatomical usage.

### Rationale

- **Alignment with text definition**: The text definition states that a squamous epithelium is "characterised by its most superficial layer consisting of squamous epithelial cells". The term implies a primary composition, not merely the presence of such cells.
- **Consistency with related issues**: A similar concern was raised in the Cell Ontology repository (obophenotype/cell-ontology#2671). Using `composed_primarily_of` is the standard pattern in Uberon for expressing that an entity is mainly made up of a particular cell type or tissue.
- **Existing pattern**: Uberon already uses `composed_primarily_of` in logical definitions for other tissues (e.g., lymphoid tissue, ommatidium), confirming this is the appropriate relation.
