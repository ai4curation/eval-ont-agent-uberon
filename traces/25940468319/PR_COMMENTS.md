# Add new term: sixth lumbar dorsal root ganglion (issue #3618)

## Summary

Adds a new UBERON term for the sixth lumbar dorsal root ganglion (L6 DRG) as requested in [issue #3618](https://github.com/obophenotype/uberon/issues/3618). The new term sits beneath the existing `lumbar dorsal root ganglion` (UBERON:0002836) alongside the existing L1–L5 sibling terms.

Rodents (e.g., mouse and rat) typically have six lumbar vertebrae, so an L6 DRG is anatomically standard in these model organisms and is widely referenced in the pain and somatosensory neuroscience literature. The request was reviewed by `@dosumis` (who confirmed the definition and the cited paper, PMID:18316160) before this work was started.

## Rationale / design choices

- **Parent**: The user-requested parent (`UBERON:0002836 lumbar dorsal root ganglion`) is the natural genus and is consistent with how the sibling L1–L5 DRGs (`UBERON:0002857`, `UBERON:0002856`, `UBERON:0002858`, `UBERON:0003943`, `UBERON:0002859`) are placed.
- **Definition**: Used verbatim from the issue (verified by `@dosumis`), cited to PMID:18316160. The phrasing mirrors the definitions on the L4 and L5 DRG sibling terms.
- **Synonyms**: Both synonyms requested in the issue were added as EXACT, matching the pattern on the L4/L5 sibling terms (e.g., `L5 dorsal root ganglion`, `fifth lumbar spinal ganglion`).
- **Subsets**: `defined_by_ordinal_series` (shared with all L1–L5 siblings) and `pheno_slim` (used on L4, L5, and the parent term).
- **No logical (intersection_of) definition**: The L1–L5 sibling terms have no logical definition either, only `is_a` to the parent. Adding an `extends_fibers_into` axiom would require a `root of L6 spinal nerve` term, which does not exist in this ontology; rather than create a second new term out of scope of the issue, the term is left without an `intersection_of` block, matching its siblings.
- **No xrefs**: Unlike L1–L5, the standard human-centric anatomy ontologies (FMA, BIRNLEX, EMAPA, UMLS) do not provide an L6 DRG class (humans have only 5 lumbar vertebrae and the EMAPA mouse mapping for L6 was not located). No xrefs were invented; xrefs can be added later if a curator identifies appropriate ones.
- **Provenance**: Contributor ORCID, `dcterms-date`, `term_tracker_item`, and `created_by: dragon-ai-agent` are set as required by `CLAUDE.md`.
