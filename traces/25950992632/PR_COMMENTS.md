# Reclassify carotid body as part_of peripheral nervous system

Fixes #3591.

## Background

The carotid body (UBERON:0001629) was previously asserted as
`part_of UBERON:0001530 ! common carotid artery plus branches`.

This is anatomically incorrect. As the issue notes, and as supported by
PMID:32965908, the carotid body is a chemoreceptive sensory organ of the
peripheral nervous system. It is *located in* the adventitia at the
bifurcation of the common carotid artery, but it is not part of the
artery — it is an adjacent, anatomically distinct structure.

The existing definition of the term already describes it as a
chemoreceptive organ, and its `is_a` parent is
`UBERON:0034979 ! nonchromaffin paraganglion`, which is consistent with
classification under the peripheral nervous system.

## Changes

Single term modified: **UBERON:0001629 carotid body**

- **Removed:** `relationship: part_of UBERON:0001530 ! common carotid artery plus branches`
- **Added:** `relationship: part_of UBERON:0000010 ! peripheral nervous system`
- **Added:** `property_value: editor_note` describing the rationale and citing PMID:32965908
- **Added:** `property_value: term_tracker_item` pointing at issue #3591

The existing `overlaps` relations to the external carotid artery
(UBERON:0001070) and internal carotid artery (UBERON:0001532) are
**retained**, so the anatomical adjacency to the carotid vasculature is
still captured in the ontology — just no longer as an incorrect
`part_of`.

The `is_a` parent (`UBERON:0034979 ! nonchromaffin paraganglion`) is
unchanged, as are all `has_part`, `capable_of`, and other relations.
