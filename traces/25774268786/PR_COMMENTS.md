# Fix taxon constraints for crustacean stomatogastric nervous system terms

## Summary
This PR fixes taxon constraint violations in 15 crustacean nervous system terms that were introduced in PR #3445.

## Problem
All affected terms had duplicate `in_taxon` assertions:
- `relationship: in_taxon NCBITaxon:6712 ! Astacidea` (lobsters)
- `relationship: in_taxon NCBITaxon:6752 ! Brachyura` (crabs)

This violates the semantics of the `in_taxon` relationship, which means "X is specific to taxon Y" - implying X cannot exist outside of Y or its descendants. Having multiple `in_taxon` assertions creates a logical contradiction.

## Solution
Replaced the dual assertions with a single assertion pointing to the common parent taxon:
- `relationship: in_taxon NCBITaxon:6692 ! Pleocyemata`

Pleocyemata is the infraorder that encompasses both Astacidea (lobsters) and Brachyura (crabs), correctly representing that these anatomical structures exist in both groups.
