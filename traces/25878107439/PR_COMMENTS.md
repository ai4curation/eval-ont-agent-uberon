# Fix incorrect classification of thoracic dorsal root ganglion

## Summary
This PR addresses issue #3475 by correcting an anatomical inconsistency in the classification of thoracic ganglia.

## Problem
Dorsal root ganglia (DRG) are sensory ganglia, while paravertebral ganglia are sympathetic ganglia. These are functionally and anatomically distinct structures that should not have a hierarchical relationship. However, the ontology incorrectly classified thoracic dorsal root ganglion (UBERON:0002835) as a subclass of thoracic ganglion (UBERON:0000961).

## Changes Made

### 1. Removed incorrect is_a relationship
**Term:** UBERON:0002835 (thoracic dorsal root ganglion)
- **Removed:** `is_a: UBERON:0000961 ! thoracic ganglion`
- **Rationale:** Dorsal root ganglia are sensory ganglia and should not be classified under paravertebral (sympathetic) ganglia
- **Retained:** The logical definition remains correct - it's still defined as a dorsal root ganglion that extends fibers into the root of thoracic nerve

### 2. Clarified term naming
**Term:** UBERON:0000961
- **Changed name from:** "thoracic ganglion" 
- **Changed name to:** "thoracic paravertebral ganglion"
- **Moved to synonym:** "thoracic ganglion" (exact synonym)
- **Rationale:** The term specifically refers to paravertebral/sympathetic ganglia, not all ganglia in the thorax. The new name clarifies this distinction and prevents future confusion.

## Anatomical Background
- **Dorsal root ganglia (DRG):** Contain sensory neuron cell bodies; located along the dorsal roots of spinal nerves
- **Paravertebral ganglia:** Part of the sympathetic nervous system; contain sympathetic neuron cell bodies; arranged in chains along the vertebral column

These are fundamentally different structures serving different functions in the nervous system, and should not be in a parent-child relationship.
